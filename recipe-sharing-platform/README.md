# Recipe Sharing Platform

A responsive recipe sharing platform built with React and Tailwind CSS.

## Features

- **Home Page**: Browse a grid of recipe cards with hover effects
- **Recipe Detail Page**: View detailed information including ingredients and cooking instructions
- **Add Recipe Form**: Submit new recipes with validation

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

## Technologies

- React
- Tailwind CSS
- React Router
- Vite

## Project Structure

- `src/components/HomePage.jsx` - Main page displaying recipe grid
- `src/components/RecipeDetail.jsx` - Individual recipe details
- `src/components/AddRecipeForm.jsx` - Form for adding new recipes
- `src/data.json` - Mock recipe data
