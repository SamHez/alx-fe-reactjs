# React JS with a CSS Framework

This project demonstrates how to integrate and use Tailwind CSS within a React application.

## Project Setup
- **Vite**: Used for scaffolding the React application.
- **Tailwind CSS**: Integrated for utility-first styling.

## Features
- **UserProfile Component**: A responsive and interactive profile card component styled with Tailwind CSS.
- **Responsive Design**: Adapts to different screen sizes.
- **Interactivity**: Hover effects and transitions.

## Usage
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
